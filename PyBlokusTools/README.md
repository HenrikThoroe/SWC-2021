# PyBlokusTools

## Install:

    sudo apt-get install -y doxygen # <- Requirement for blokusdoc
    python3.8 -m pip install --upgrade . # <- In PyBlokusTools folder
  
## Available Scripts:
  

 - blokustest
 - blokusbench
 - blokusprod
 - blokusdebug
 - blokusdoc

## Building Documentation:

The docs are build from `docsrc/source/`
